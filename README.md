# iBrood PWA - Queen Cell & Brood Pattern Analysis

A Progressive Web App for beekeepers to analyze queen cell maturity and brood patterns using AI-powered image analysis.

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Installation

1. Install dependencies:
```bash
npm install
# or
pnpm install
```

2. Run the development server:
```bash
npm run dev
# or
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Features

- **Queen Cell Analysis**: Upload or capture images to analyze queen cell maturity
- **Brood Pattern Analysis**: Monitor hive health through brood pattern recognition
- **Analytics Dashboard**: Track trends and hive performance over time
- **PWA Support**: Install as a mobile app for field use
- **Responsive Design**: Works on desktop, tablet, and mobile devices
